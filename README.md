## Description

> A server-side application that allows you to add a user and tracks their exercise progress.

<br />

## Setup

> - Create a new `.env` file and put the corresponding `key-value` pairs.

```bash
# * MongoDB
MONGODB_URI=

# * Local
PORT=
```

> - Run the following commands.

```bash
npm run build
npm run dev
```
